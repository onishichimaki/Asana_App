    Known risks / unknowns

    事実と分離
    !
    本格認証なし

    外部公開前に組織認証、TLS、rate limitが必要。

    ?
    Gemini実通信待ち

    実装とモックテストは完了。露出キーを失効・再発行して実APIスモークが必要。

    ?
    実連携検証はローカル限定

    SQL ServerとAsana限定projectへの実登録は成功。HTTPS配備と運用監視は未検証。

    ?
    音声入力のブラウザー差

    Web Speech APIとHTTPS、マイク権限に依存。

    ?
    OCR初回モデル取得

    初回のみ日本語モデル取得が必要。精度は画像品質に依存し、確認・修正を前提とする。
